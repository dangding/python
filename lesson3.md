### 变量

在编程语言中，变量名相当于人的名字，是一个标识。变量的命名必须符合[上一章](lesson2.md)所介绍的规则。

在python中，变量不需要进行声明，在变量定义时，不需要指定该变量用来表示哪种数据类型，在对变量进行赋值时，python会自动识别。

比如：

```
a = "abcd"   # a是字符串类型变量
b = 123      # b是整型变量
c = 123.456  # c是浮点变量
```

### 类型

python中所支持的数据类型主要有：

#### 数字

python支持的数字类型有：int(有符号整型)，float(浮点型)，long(长整型)和complex(复数)。

数字相关的运算符有：`+、-、*、/`，也可以利用`()`来分组。

```
加法
>>> 2 + 3
5

减法
>>> 6 - 5
1

乘法
>>> 2 * 4
8

除法，返回浮点类型
>>> 5 / 2
2.5

整除
>>> 5 // 2
2

乘方
>>> 3 ** 2
9

括号优化级
>>> (10 - 2 * 3) / 3
1.3333333333333333
```

#### 字符串

字符串就是，用单引号或双引号所包裹的内容。如：`abcd`、`efgh123`

字符串的存储是一种数组结构，下标索引从0开始。如字符串“abcdef”的存储结构为：

|  a   |  b   |  c   |  d   |  e   |  f   |
| :--: | :--: | :--: | :--: | :--: | :--: |
|  0   |  1   |  2   |  3   |  4   |  5   |

基本操作

```
字符串连接
>>> "aa"+"bb"
'aabb'

对变量s进行赋值
>>> s = "abcdef"

通过下标引用字符串
>>> s[0]
'a'

下标可以是负数，表示从末尾开始，-1代码最后一个
>>> s[-1]
'f'

截取下标[0,3)的字符串
>>> s[0:3]
'abc'

从下标2开始截取字符串，直至结尾
>>> s[2:]
'cdef'

内置函数求字符串长度
>>> len(s)
6
```

#### 列表

列表是通过方括号括起，逗号分隔的一组元素。一个列表中可以包含不同类型的元素，但通常使用时各个元素的数据类型相同。

列表的存储结构与字符串一样使用数组形式存储，其下标也是从0开始递增。

```
初始化赋值
>>> l = [1, 2, 3, 4, 5]
>>> l[0]
1

修改列表中下标为2的元素
>>> l[2]=11
>>> l
[1, 2, 11, 4, 5]

从末尾下标引用
>>> l[-1:]
[5]

截取从下标3开始的元素
>>> l[3:]
[4, 5]

连接两个列表
>>> m = [6, 7, 8]
>>> l + m
[1, 2, 3, 4, 5, 6, 7, 8]

求列表中元素个数
>>> len(l)
5

向列表尾部追加元素
>>> m.append(10)
>>> m
[6, 7, 8, 10]
```

#### 元组

元组是由几个逗号分隔的值组成的数据类型，其存储结构也与列表相同，其与列表的区别是，不能对元素进行修改。

基本操作有：

```
初始化元组
>>> t=(1,2,3,4,5,6)

引用元素
>>> t[0]
1
>>> t[-1]
6

元素切片
>>> t[2:]
(3, 4, 5, 6)

试图修改元素值，会抛出异常
>>> t[0]=12
Traceback (most recent call last):
  File "<pyshell#59>", line 1, in <module>
    t[0]=12
TypeError: 'tuple' object does not support item assignment

统计元组数据个数
>>> len(t)
6
```

#### 字典

字典类型，实际上是一种表示，`键-值`对的数组结构。通过情况下，键的数据类型为数字或字符串。

基本操作

```
字典类型赋值
>>> d = {"name":"zhangsan", "age": 35, "gender":"F"}
>>> print(d)
{'name': 'zhangsan', 'age': 35, 'gender': 'F'}

引用元素
>>> d["name"]
'zhangsan'

计算元素个数
>>> len(d)
3

引用一个不存在的key，会抛出异常
>>> d["address"]
Traceback (most recent call last):
  File "<pyshell#68>", line 1, in <module>
    d["address"]
KeyError: 'address'
```



### 运算符

#### 算术运算

以下假设变量：a=10，b=20

| 运算符 | 描述                           | 实例                |
| ------ | ------------------------------ | ------------------- |
| +      | 两个对象相加                   | a + b输出结果 30    |
| -      | 表示负数或都一个数减少另一个数 | a - b输出结果 -10   |
| *      | 两个数相乘                     | a * b 输出结果是200 |
| /      | x 除以 y                       | b / a 输出结果是2   |
| %      | 取模，即返回除法的余数         | b % a 输出结果是0   |
| **     | 幂，返回x的y次幂               | a**b 表示10的20次方 |
| //     | 取整除，返回商的整数部分       | 9 // 2，结果为4     |

#### 关系运算

以下假设变量：a=10, b=20

成立则返回`True`，不成立则返回`False`

| 运算符 | 描述                   | 实例              |
| ------ | ---------------------- | ----------------- |
| ==     | 比较对象是否相符       | (a == b)返回False |
| !=     | 比较两个对象是否不相等 | (a != b)返回True  |
| >      | 返回x是否大于y         | (a > b)返回False  |
| <      | 返回x是否小于y         | (a < b)返回True   |
| >=     | 返回x是否大于等于y     | (a >= b)返回False |
| <=     | 返回x是否小于等于y     | (a <= b)返回True  |

需要注意的是，编程语言中，所有的非0，都表示`True`。

#### 逻辑运算

| 运算符 | 逻辑表达式 | 描述                                                   | 实例                                                         |
| ------ | ---------- | ------------------------------------------------------ | ------------------------------------------------------------ |
| and    | x and y    | 与运算：只x和y同为True时，才返回True，其他返回False    | >>> x = True;y = False<br/>>>> x and y<br/>False<br/><br/>>>> x = True; y = True<br/>>>> x and y<br/>True<br/><br/>>>> x = False; y = True<br/>>>> x and y<br/>False |
| or     | x or y     | 或运算：只有x和y同为False时，才返回False，其他返回True | >>> x = True; y = False<br/>>>> x or y<br/>True<br/><br/>>>> x = False; y = False<br/>>>> x or y<br/>False<br/><br/>>>> x = True; y = True<br/>>>> x or y<br/>True<br/><br/>>>> x = False; y = True<br/>>>> x or y<br/>True |
| not    | not x      | 非运算：如果x为True，返回False;若x为False，则返回True  | >>> x = True<br/>>>> not x<br/>False<br/><br/>>>> x = False<br/>>>> not x<br/>True |

#### 成员运算

| 运算符 | 描述                                                | 实例                                                         |
| ------ | --------------------------------------------------- | ------------------------------------------------------------ |
| in     | 如果在指定的序列中找到值，返回True，否则返回False   | >>> a = [1, 2, 3, 4]<br/>>>> 2 in a<br/>True<br/><br/>>>> 5 in a<br/>False |
| not in | 如果在指定序列中没有找到值，返回True，否则返回False | >>> a = [1, 2, 3, 4]<br/>>>> 3 not in a<br/>False<br/><br/>>>> 5 not in a<br/>True |
