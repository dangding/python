### 变量

在编程语言中，变量名相当于人的名字，是一个标识。变量的命名必须符合[上一章](lesson2.md)所介绍的规则。

在python中，变量不需要进行声明，在变量定义时，不需要指定该变量用来表示哪种数据类型，在对变量进行赋值时，python会自动识别。

比如：

```
a = "abcd"   # a是字符串类型变量
b = 123      # b是整型变量
c = 123.456  # c是浮点变量
```
